# 资料分析

## 常用术语

| 术语分类       | 术语名称       | 核心定义/公式                                                                 | 关键提示                                                                 |
|----------------|----------------|------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| **时间对比**   | 同比           | 与历史同一时期（通常上年同期）比较                                           | 例：2023年2月 vs 2022年2月                                               |
|                | 环比           | 与上一个统计周期（年/月/日）比较                                             | 例：2023年2月 vs 2023年1月                                               |
| **增长相关**   | 基期           | 对比参照的时期（时间在前）| 跟谁比，谁就是基期                                                       |
|                | 现期           | 相对于基期的时期（时间在后）| —                                                                          |
|                | 增长量         | 现期值 - 基期值（绝对差值）| 正增长为增量，负增长为减量                                               |
|                | 增长率         | 增长量÷基期值×100%（相对比值）| 也叫增速、增幅                                                           |
|                | 年均增长量     | （末期量 - 初期量）÷年数                                                     | 年数为末期与初期的时间差                                                 |
|                | 年均增长率     | 末期量 = 初期量×(1+年均增长率)ⁿ（n为年数）| 计算常需结合近似值或公式变形                                             |
|                | 幅度           | 现期与基期增减的比率                                                         | 比较涨幅大小看绝对值                                                     |
| **比例相关**   | 比重           | 部分量÷整体量（百分数形式）| 整体量=部分量÷比重；部分量=整体量×比重                                   |
|                | 百分数         | 分母为100的分数                                                               | 表示比例或增长率                                                         |
|                | 百分点         | 百分数的差值单位，1个百分点=1%                                               | 用于描述增长率、比重的变化                                               |
| **数据关系**   | 平均数         | 总数÷份数                                                                     | 反映数据平均水平                                                         |
|                | 倍数           | 两个数据的对比数（A是B的几倍）| 倍数=A÷B                                                                 |
|                | 成数           | 1成=10%，几成即十分之几                                                       | 例：2成=20%                                                              |
|                | 翻番           | 翻n番=变为原来的2ⁿ倍                                                          | 例：翻2番=4倍，翻3番=8倍                                                 |
| **经贸相关**   | 顺差           | 出口额 > 进口额                                                               | 贸易盈余                                                                 |
|                | 逆差           | 进口额 > 出口额                                                               | 贸易赤字                                                                 |
| **发展相关**   | 发展速度       | 现期÷基期=1+增长率                                                            | 反映发展的相对速度                                                       |
|                | 拉动增长率     | 部分增长量÷整体基期量                                                         | 衡量部分对整体增长的拉动作用                                             |
|                | 增长贡献率     | 部分增长量÷整体增长量                                                         | 衡量部分对整体增长的贡献程度                                             |
| **民生系数**   | 恩格尔系数     | 食品支出总额÷消费支出总额                                                     | 系数越低，生活水平越高                                                   |
|                | 基尼系数       | 反映收入分配差距，取值0-1                                                     | 系数越大，收入不平等程度越高                                             |

## 速算技巧

一、基础速算类

| 技巧名称       | 适用场景                                                                 | 核心方法                                                                 |
|----------------|--------------------------------------------------------------------------|--------------------------------------------------------------------------|
| **直算法**     | 题目运算简单、选项差异极小（需精确计算）                                 | 老老实实做基础加减乘除，保证精度。 |
| **尾数法**     | 求多个数的和/差，且选项尾数差异大                                       | 只取各数的尾数进行运算，通过结果尾数锁定答案。 |
| **截位法**     | 选项精度要求不高、数字差距大                                             | 四舍五入截取数字前几位（如前2位），简化后再计算。 |
| **位数求和错位相加法** | 多个数求和，需精确结果但不想复杂进位                                   | 从高位到低位逐位求和，再将“和”错位相加，简化进位计算。 |


二、巧算简化类

| 技巧名称       | 适用场景                                                                 | 核心方法                                                                 |
|----------------|--------------------------------------------------------------------------|--------------------------------------------------------------------------|
| **标尺法**     | 多个数求和/求平均，数字彼此接近                                           | 选一个基准数（“标尺”），先算标尺×个数，再对每个数与标尺的差值做修正。 |
| **分段法**     | 三位/四位减法，避免借位运算                                               | 将减数拆成两段，分别相减，避免“借位”。 |
| **分数法**     | 百分数/复杂小数运算                                                       | 将百分数转化为常见分数（如20%=1/5，12.5%=1/8），简化乘除。 |
| **拆分法**     | 除法运算，结果在100%/50%/1%等附近                                       | 将大数拆成便于计算的小数（如450=484-34），再分别运算。 |
| **放缩法**     | 选项差距大，只需判断大小关系                                               | 适当放大/缩小数字，快速比较选项间的大小。 |
| **份数法**     | 已知增长率/变化量，需计算基期/增长量                                     | 将百分数转化为份数（如16.61%≈1/6），用“份数”代表基期/增长量。 |
| **分配法**     | 已知现期和增长率，求增长量（增长率接近整分数）                           | 将现期拆为“基期+增长量”，按分数比例分配，快速估算增长量。 |


三、核心使用原则

1.  **先看选项**：根据选项差距决定用“精算”还是“估算”。
2.  **优先简化**：能转分数、拆数、用标尺的，尽量避免硬算。
3.  **精度优先**：选项差距小时，必须用直算法保证结果准确。
