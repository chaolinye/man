# 言语理解与表达

## 核心原则

- 不要过度追求“完美答案”，优先贴合作者写作习惯与原文逻辑。
- 语感优先，技巧为辅；弱语感阶段用技巧辅助，强语感阶段回归直觉。

## 二、高频解题方法

| 方法类别               | 核心要点                                                                 |
|------------------------|--------------------------------------------------------------------------|
| **关注高频词**         | 材料中反复出现的名词/短语是核心话题，引导文段重点。                       |
| **关注关联词**         | - **表转折**：但是/然而/却 → 转折后是重点<br>- **表递进**：不但/而且/甚至 → 递进后更重要<br>- **表因果**：因为/所以/因此 → 结果是重点<br>- **表并列**：既/又/同时 → 内容同等重要<br>- **表条件**：如果/只要/只有 → 条件是核心 |
| **关注文章结构**       | 常见结构：总-分、分-总、总-分-总、并列、分-分 → 抓总起/总结句找主旨。     |
| **关注标点符号**       | 顿号/分号表并列；冒号/破折号表解释；问号表反问/设问；引号表强调/引用。     |
| **关注代词**           | 这/那/它/他们等指代前文信息，就近指代，简化阅读。                         |
| **避免偷换与无中生有** | 警惕偷换主体、数量、时态、语气；无中生有是选项中原文未提及的内容。         |
| **关注语义轻重**       | 如“随口胡说”比“信口开河”语义更重，需匹配原文语气。                         |
| **关注感情色彩**       | 褒贬词区分作者态度，如“处心积虑”（贬）与“殚精竭虑”（褒）。                 |
| **关注句子成分**       | 主谓宾定状补搭配要合理，避免搭配不当、成分残缺/冗余。                     |
| **关注观点位置**       | 观点常前置或后置，引入话题后再抛出核心观点。                               |
| **关注事物发展顺序**   | 排序题优先遵循时间/逻辑/自然顺序（如春夏秋冬）。                           |
| **关注主旨句**         | 三种归纳方式：<br>1. 直接找原文主旨句<br>2. 主体+措施+结果<br>3. 主谓宾结构 |

## 三、题型对应思路

| 大纲要求                               | 对应题型               | 解题思路                     |
|----------------------------------------|------------------------|------------------------------|
| 查找主要信息/细节                       | 细节理解               | 定位原文，警惕偷换/无中生有   |
| 概括中心/主旨                           | 概括主旨/选标题         | 抓高频词、主旨句、文章结构     |
| 判断语句与原文是否一致                   | 语句填空/排序           | 关注逻辑、代词、标点、顺序     |
| 推断隐含信息                           | 推断下文内容           | 顺承尾句话题，遵循逻辑顺序     |
| 判断作者态度/意图                       | 判断意图/理解态度       | 看感情色彩、关联词、主旨句     |
| 理解指定词语/语句含义                   | 词语指代               | 就近指代，结合语境理解         |
| 准确得体遣词用字                       | 逻辑填空               | 看语义轻重、感情色彩、搭配     |
