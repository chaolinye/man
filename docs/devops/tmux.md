# Tmux

## 一、核心概念模型
Tmux 是**终端会话复用器**，核心是**会话与窗口解绑**，关闭窗口不终止进程，支持多会话、多窗口、多窗格分层管理。

### 1. 三层结构
- **会话（Session）**：最顶层，一组窗口的集合，后台持久运行，可分离/重入
- **窗口（Window）**：会话内的独立全屏终端，一个会话可开多个窗口
- **窗格（Pane）**：窗口内拆分的小终端，一个窗口可拆分成多个窗格

```
+----------------+
|  TmuxServer    |
+----------------+
| - sessions: Session[] |
+----------------+
         | 1..*
         |
+----------------+
|    Session     |
+----------------+
| - name: string |
| - windows: Window[] |
+----------------+
         | 1..*
         |
+----------------+
|    Window      |
+----------------+
| - index: int   |
| - title: string|
| - panes: Pane[]|
+----------------+
         | 1..*
         |
+----------------+
|     Pane       |
+----------------+
| - index: int   |
| - pid: int     |
| - command: string |
+----------------+
```

### 2. 核心作用
- 窗口关闭，会话与进程**后台继续运行**（解决SSH断连丢失任务）
- 单终端同时管理**多个会话/窗口/窗格**
- 支持**会话共享**与**窗格自由拆分**

---

## 二、通用规则
- 前缀键：**Ctrl+b**（所有快捷键必须先按它）
- 退出帮助：**q / ESC**

---

## 三、常用快捷键（按模块分类）
### 1. 会话管理（Session）
- `Ctrl+b d`：分离当前会话（后台保留）
- `Ctrl+b s`：列出并切换会话
- `Ctrl+b $`：重命名当前会话

### 2. 窗格操作（Pane）
- `Ctrl+b %`：左右分屏
- `Ctrl+b "`：上下分屏
- `Ctrl+b 方向键`：切换窗格
- `Ctrl+b z`：当前窗格全屏/恢复
- `Ctrl+b x`：关闭当前窗格
- `Ctrl+b q`：显示窗格编号
- `Ctrl+b Ctrl+方向键`：调整窗格大小

### 3. 窗口管理（Window）
- `Ctrl+b c`：新建窗口
- `Ctrl+b n`：下一个窗口
- `Ctrl+b p`：上一个窗口
- `Ctrl+b 数字`：切换到对应编号窗口
- `Ctrl+b w`：从列表选窗口
- `Ctrl+b ,`：重命名窗口

---

## 四、最常用命令（速记）
- 新建命名会话：`tmux new -s 名称`
- 查看会话：`tmux ls`
- 重入会话：`tmux attach -t 名称/编号`
- 杀死会话：`tmux kill-session -t 名称/编号`
