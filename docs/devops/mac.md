# Macbook 教程

## 一、首先，搞懂 Mac 的“神奇修饰键”

在开始之前，必须认识这几个符号。它们比字母本身更重要：

| 符号 | 对应按键 | Windows 类比 |
| :--- | :--- | :--- |
| ⌘ | Command | Ctrl (大部分情况) |
| ⌥ | Option | Alt |
| ⇧ | Shift | Shift |
| ⌃ | Control | Ctrl (少数情况) |
| ⇪ | Caps Lock | 大写锁定 |

> 💡 **核心心法**：在 Windows 里复制是 `Ctrl + C`，在 Mac 里就是 `⌘ + C`。**把肌肉记忆里的 `Ctrl` 换成 `⌘`**，你就入门了 80%。

## 二、必备基础篇（每天用 100 次）

这些是保命快捷键，请务必烂熟于心。

| 操作 | 快捷键 | 备注 |
| :--- | :--- | :--- |
| **复制 / 粘贴 / 剪切** | `⌘ + C` / `⌘ + V` / `⌘ + X` | 注意：**剪切文件**是 `⌘ + Option + V` |
| **撤销 / 重做** | `⌘ + Z` / `⌘ + Shift + Z` | 和 Windows 顺序不同，这里重做多了 Shift |
| **保存** | `⌘ + S` | 养成习惯，每秒按一次 |
| **全选** | `⌘ + A` | 同上 |
| **关闭当前窗口** | `⌘ + W` | 只关标签页或窗口，不退出软件 |
| **彻底退出软件** | `⌘ + Q` | 关掉后台。这是 Mac 特色，别按错了 |
| **隐藏当前软件** | `⌘ + H` | 瞬间退回桌面，软件还在后台 |
| **切换应用** | `⌘ + Tab` | 按住 ⌘，按 Tab 选择，松开切换 |
| **强制退出应用** | `⌘ + Option + Esc` | 相当于 Windows 的任务管理器 |

## 三、截图与录屏（Mac 最强，没有之一）

Mac 自带的截图功能，强大到不需要装任何第三方工具。

| 操作 | 快捷键 | 效果 |
| :--- | :--- | :--- |
| **全屏截图** | `⌘ + Shift + 3` | 整个屏幕拍下来，保存到桌面 |
| **区域截图** | `⌘ + Shift + 4` | 十字光标出现，框选区域截图 |
| **窗口截图** | `⌘ + Shift + 4` + 空格 | 光标变相机，点击任意窗口截图（带阴影） |
| **全屏录屏** | `⌘ + Shift + 5` | 调出录屏控制栏，可选项很多 |

> ⚡ **进阶技巧**：按住 `Ctrl` 再按上述组合（比如 `⌘ + Ctrl + Shift + 4`），截图会直接存入**剪贴板**，而不是桌面。适合粘贴到聊天窗口。

## 四、访达（Finder）里的生存技巧

访达就是 Mac 的“我的电脑”，用好快捷键效率翻倍。

| 操作 | 快捷键 | 说明 |
| :--- | :--- | :--- |
| **新建窗口** | `⌘ + N` | 打开一个新访达窗口 |
| **新建文件夹** | `⌘ + Shift + N` | 在当前目录创建文件夹 |
| **显示/隐藏文件** | `⌘ + Shift + .` | 显示隐藏文件（极常用） |
| **前往当前用户文件夹** | `⌘ + Shift + H` | 快速到你的“家目录” |
| **前往电脑根目录** | `⌘ + Shift + C` | 相当于“此电脑” |
| **前往文件夹** | `⌘ + Shift + G` | 手动输入路径跳转 |
| **删除文件** | `⌘ + Delete` | 移到废纸篓 |
| **彻底删除** | `⌘ + Option + Delete` | 不经过废纸篓，直接删除 |
| **文件预览** | `空格键` | **Mac 神技**：选中文件按空格，瞬间预览内容 |

## 五、浏览器与文本编辑（效率翻倍区）

这部分是很多 Mac 用户用了三年都不知道的宝藏。

| 操作 | 快捷键 | 效果 |
| :--- | :--- | :--- |
| **光标移动到行首/行尾** | `⌘ + 左箭头` / `⌘ + 右箭头` | 比 Home/End 快多了 |
| **光标移动到文档开头/结尾** | `⌘ + 上箭头` / `⌘ + 下箭头` | 阅读长文必备 |
| **删除光标后一个字** | `Fn + Delete` | 相当于 Windows 的 Delete 键 |
| **删除光标到行尾的内容** | `⌘ + Delete` | 一键清空当前行后半段 |
| **浏览器新建标签页** | `⌘ + T` | 无限刷网页 |
| **关闭当前标签页** | `⌘ + W` | 关掉不想看的 |
| **重新打开关闭的标签页** | `⌘ + Shift + T` | 手滑后的后悔药 |
| **浏览器地址栏** | `⌘ + L` | 光标跳转到网址栏，直接输入搜索 |
| **在多个标签页间切换** | `⌘ + Tab` 或 `⌘ + Shift + 左右括号` | 后者更直观 |

## 六、窗口与多任务管理（告别鼠标拖拽）

Mac 没有 Windows 那种“拖到边缘自动半屏”的功能？有的，而且更好用。

| 操作 | 快捷键 | 说明 |
| :--- | :--- | :--- |
| **进入/退出全屏** | `⌘ + Ctrl + F` | 或者点窗口绿点。全屏后是一个独立桌面 |
| **同一应用多窗口切换** | `⌘ + `` ` (反引号，Tab上面那个键) | 比如开了 5 个 Chrome 窗口，用这个切 |
| **调度中心（看所有窗口）** | `⌃ + 上箭头` 或三指上划 | 一览所有桌面和窗口 |
| **显示桌面** | `⌘ + F3` 或四指张开 | 所有窗口弹开，露出桌面 |
| **左右切换桌面** | `⌃ + 左/右箭头` 或三指左右滑 | 如果你有多个桌面（Space） |
| **当前应用隐藏其他** | `⌘ + Option + H` | 只留当前软件，其他全隐藏，极度专注 |

## 七、启动与搜索（Mac 的灵魂）

Mac 最反直觉也最高效的功能：**你不用满桌面找图标**。

- **Spotlight 聚焦搜索**：`⌘ + 空格`
  - 输入 App 名字回车 → 打开软件
  - 输入计算式 → 直接出结果
  - 输入文件名 → 瞬间定位文件
  - 输入汇率/翻译 → 秒出答案

> 自从用了 Spotlight，我的 Launchpad（启动台）几乎没打开过。

## 八、触控板手势（不用学，靠直觉）

Mac 触控板是独立于快捷键的另一套效率体系。默认开启，强烈建议不要关。

| 手势 | 功能 |
| :--- | :--- |
| **单指轻点** | 单击 |
| **双指轻点** | 右键菜单 |
| **双指上下滑动** | 滚动页面（和手机一样） |
| **双指捏合** | 放大缩小 |
| **三指拖移** | 移动窗口或选中文字（需在设置里开启） |
| **四指上推** | 调度中心 |
| **四指左右扫** | 切换桌面 |
| **四指抓拢** | 启动 Launchpad |

## 九、Mac 用户最容易搞错的几个坑（重点）

1. **为什么我“剪切”不了文件？**  
   Mac 的逻辑是：先 `⌘ + C` 复制，再到目标位置按 `⌘ + Option + V` = 移动（剪切）。直接 `⌘ + X` 对文件无效。

2. **红点（关闭）为什么没退出软件？**  
   红点只是关窗口，软件还在后台。想要彻底退出用 `⌘ + Q`。这是为了让你下次打开更快。

3. **为什么没有“我的电脑”和 C 盘？**  
   访达（Finder）就是。按 `⌘ + Shift + C` 就能看到磁盘根目录。

4. **怎么强制重启/关机？**  
   按 `⌃ + ⌘ + 电源键` 强制重启。不到万不得已别用。
