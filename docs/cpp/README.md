# C++

从 2011 年发布的 `C++11` 开始，C++ 标准委员会承诺每 3 年发布一个新的 C++ 标准。截止到目前，已经发布了 `C++14`、`C++17`、`C++20` 标准以及若干个扩展 C++ 标准的技术规范。
