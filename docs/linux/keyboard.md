# 快捷键

`Bash` 内置了 `Readline` 库，具有这个库提供的很多“行操作”功能，比如命令的自动补全，可以大大加快操作速度。

!> 除了 Bash，许多 CLI 工具默认也使用

这个库默认采用 `Emacs` 快捷键，也可以改成 `Vi` 快捷键。

```bash
# 切换成 vi 快捷键
set -o vi
# 切换成 emacs 快捷键
set -o emacs
```

> 本文介绍的快捷键都属于 `Emacs` 快捷键

| Action | Keyboard |
| :--: | :--: |
| 移动到行首 | `Ctrl + a` |
| 移动到行尾 | `Ctrl + e` |
| 左移一个字符 | `Ctrl + b` |
| 右移一个字符 | `Ctrl + f` |
| 移动到词尾 | `Alt + f` |
| 移动到词首 | `Alt + f` |
| 删除光标前面的单词 | `Ctrl + w` |

> 上面快捷键的 Alt 键，也可以用 ESC 键替代

## 光标移动
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + a` | 移动到行首 |
| `Ctrl + e` | 移动到行尾 |
| `Ctrl + f` | 向右移动一个字符 |
| `Ctrl + b` | 向左移动一个字符 |
| `Alt + f` | 向右移动一个单词 |
| `Alt + b` | 向左移动一个单词 |
| `Ctrl + xx` | 在当前位置与行首之间切换 |

## 文本编辑
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + h` | 删除光标前一个字符（同 Backspace） |
| `Ctrl + d` | 删除光标所在字符；若行为空则退出当前 shell |
| `Ctrl + w` | 删除光标前的一个单词（以空格/标点分隔） |
| `Alt + d` | 删除光标后的一个单词 |
| `Ctrl + u` | 删除从光标到行首的所有字符 |
| `Ctrl + k` | 删除从光标到行尾的所有字符 |
| `Ctrl + y` | 粘贴最近一次删除的文本（yank） |
| `Alt + y` | 在 `Ctrl + y` 后使用，轮换到更早的删除记录 |
| `Ctrl + _` 或 `Ctrl + x, u` | 撤销上次编辑操作 |
| `Ctrl + t` | 交换光标前后两个字符 |
| `Alt + t` | 交换光标前后两个单词 |
| `Alt + u` | 将光标后单词转为大写 |
| `Alt + l` | 将光标后单词转为小写 |
| `Alt + c` | 将光标后单词首字母大写 |

## 历史命令
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + p` | 上一条命令（同 ↑） |
| `Ctrl + n` | 下一条命令（同 ↓） |
| `Ctrl + r` | 反向增量搜索历史（继续按 `Ctrl + r` 查找更早匹配） |
| `Ctrl + s` | 正向增量搜索历史（可能被终端流控占用，需 `stty -ixon` 解除） |
| `Alt + .` 或 `Alt + _` | 插入上一命令的最后一个参数（重复按可轮换更早的命令） |

## 控制与杂项
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + l` | 清屏（相当于 `clear`） |
| `Ctrl + c` | 中断当前正在运行的命令或取消当前输入的行 |
| `Ctrl + z` | 挂起当前前台进程（可用 `fg` 恢复） |
| `Ctrl + s` / `Ctrl + q` | 暂停/恢复终端输出（流控制，如被占用可禁用） |
| `Tab` | 自动补全命令、文件名或变量 |
| `Ctrl + o` | 执行当前命令，并自动从历史中取出下一条命令（用于重复执行多条历史命令） |
