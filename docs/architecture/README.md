# 架构设计

## 架构师的主要工作

- 功能分析(画框框)

- 技术选型(挑工具)

- 架构设计(高内聚/低耦合)

## 架构原则

> 计算机科学是这样一门学科：它相信所有问题都可以通过添加一个间接层来解决
